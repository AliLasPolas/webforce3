Valeurs

Observez le programmevaleurs.js et tentez de prévoir les valeurs finales de chaque variable.

var a = 2;

a = a - 1;

a++;

var b = 8;

b += 2;

var c = a + b * b;

var d = a * b + b;

var e = a * (b + b);

var f = a * b / a;

var g = b / a * a;

Vérifiez vos prévisions en créant ce programme puis en le déboguant pour observer les valeurs des variables.

a 2

b 10

c 102
